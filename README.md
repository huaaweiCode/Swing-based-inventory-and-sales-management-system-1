## 1、项目介绍

基于swing的进存销管理系统实现了商品管理、入库管理、出库管理、销售单管理、仓库管理等功能


## 2、项目技术

前端技术：swing

## 3、开发环境

- JAVA版本：JDK1.8
- IDE类型：IDEA、Eclipse都可运行
- 数据库类型：MySql（5.x和8.x版本都可） 
- 硬件环境：Windows 或者 Mac OS
